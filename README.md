# 🎬 FlixTrack - React Movie Management System

A modern and responsive **Movie Management System** built with **React.js**. The application allows users to perform **Create, Read, Update, Delete (CRUD)** operations on movies and provides instant search functionality with a clean user interface.

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/balajiinfo8/React-MovieRecommendationApp.git
```

### Navigate to the project

```bash
cd React-MovieRecommendationApp
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Open in your browser

```
http://localhost:5173
```
